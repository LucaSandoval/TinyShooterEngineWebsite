Detailed Description

The ApplicationV2 class represents an instance of a TinyShooter Engine application. Manages SDL to provide a game winodw to use, manages loading and configuring scenes to be loaded and used in the game, and handles input for the game. Provides a public Tick() method to control the passage of each frame in the engine.
