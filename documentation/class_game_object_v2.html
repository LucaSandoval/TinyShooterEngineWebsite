Detailed Description

GameObjectV2 is a class representing a GameObject in TinyShooter Engine. GameObjects are the most basic object in a TinyShooter Engine game, capable of having several different ComponentV2 attatchments that provide its functionality. All GameObjectV2 have a TransformComponentV2 by default.
